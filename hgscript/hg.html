💡 HGScript は Discord Bot を直感的に記述するための DSL です。入力したコードはリアルタイムで discord.py 2.x へ変換されます。

      
基本ルール

      
ブロックは { } で囲みます。文は改行で区切ります。コメントは # から始まります。

      
# コメント
command(hello) {
    reply "Hello!"
}

コマンドの定義

      
スラッシュコマンドは command(名前: 引数) で定義します。

      
command(greet: string name) {
    reply "Hi, " + name
}

      
引数の種類

      

        
構文説明Python 変換

        
type name必須引数name: int

        
type? name省略可能（None 許容）name: int = None

      

      
# 複数引数
command(add: int a int b) {
    int result = a + b
    reply result
}
# 省略可能引数
command(hello: string? name) {
    if name == "" {
        reply "Hello!"
    }
    else {
        reply "Hello, " + name
    }
}

store / load

      
Key-Value 形式でデータを保存・読み込みます。現在の実装はメモリ上の dict です。本番環境では SQLite や Redis などへの置き換えを推奨します。

      
# 保存
store coins = 500
# 読み込み（存在しない場合は 0 が返る）
let coins = load coins
# 加算して更新
int newCoins = coins + 100
store coins = newCoins

      
await fetch — 外部 API

      
# 結果を変数に受け取る
let data = await fetch "https://api.example.com/data"
# 結果を使わずリクエストだけ送る
await fetch "https://api.example.com/ping"

      
💡 await fetch は内部で aiohttp に変換されます。
import aiohttp が自動的に出力コードへ追加されます。

      
Economy コマンド例

      
command(earn) {
    cooldown 3600
    let bal = load coins
    int reward = 100
    int total = bal + reward
    store coins = total
    reply "💰 " + reward + " coins 獲得！"
}
